# Piyasa Riski Modüler Hesaplama Toolu

Faiz, nakit akışı, bono, getiri eğrisi, türev ürün, hedge, durasyon,
konveksite ve VaR başlıklarını tek bir mobil uyumlu web uygulamasında toplayan
statik hesaplama aracı.

## Kapsam

- 36 ayrı piyasa riski ve finansal matematik modülü
- Her modülde açıklama, gerçek finansal formül, değişken sözlüğü ve canlı sonuç
- Arama ve kategori filtresi
- Mobil uyumlu beyaz tema
- Statik web yayınına uygun sade dosya yapısı

## Yerel Kullanım

`index.html` dosyasını tarayıcıda açman yeterli.

## Netlify

Bu proje statiktir. GitHub reposunu Netlify'a bağladığında ayarlar
`netlify.toml` dosyasından otomatik okunur.

- Build command: boş bırakılabilir
- Publish directory: `.`
